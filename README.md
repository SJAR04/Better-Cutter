## Overview
Better Cutter is a datapack that overhauls Minecraft Stonecutter, enabling it to craft wooden items like a dedicated woodworking station. Designed for players who want streamlined building without mods, it retains vanilla balance while expanding the stonecutter‚Äôs utility for all 6 classic wood types (oak, spruce, jungle, birch, dark oak, acacia).

## Features
Stonecutter Woodworking:

Craft planks, slabs, stairs, fences, doors, trapdoors, and signs directly in the stonecutter.

Vanilla-friendly ratios (e.g., 1 log = 4 planks, 1 log = 4 stairs).

Customizable Recipes:

Optional config to disable vanilla crafting table recipes (e.g., force fences to be stonecutter-only).

Adjust plank/stair yields for balance.

Optimized Performance:

Minimal lag during /reload (v3.4 improvements).

No visual glitches (axes no longer ‚Äúconsumed‚Äù).

## Compatibility
Works With:

Fabric, Forge, or Vanilla (no mods required!).

Most resource packs and performance mods (e.g., Sodium, OptiFine).

Conflicts:

Datapacks/mods that alter vanilla stonecutter behavior (e.g., Terralith, Vanilla Tweaks).

## FAQ
Q: Do all vanilla wood recipes work in the stonecutter?\
A: Yes! All 6 wood types (oak, spruce, jungle, birch, dark oak, acacia) are supported.

Q: Can I disable certain recipes?\
A: Yes! Edit the config file to toggle specific recipes or yields.

Q: Does this work with Fresh Animations or other resource packs?\
A: Yes, but use the optional texture pack for stonecutter GUI hints.

Q: Backports to newer versions?\
A: This version is exclusive to 1.13.X. For 1.15+, check our updated packs!

## Limitations
No support for post-1.14 blocks (e.g., Bamboo, Crimson).

Recipe book previews may require a /reload to update.
